INTRODUCCION AL DISEÑO DE SITIOS WEB

OBJETIVO

El curso diseño de sitios web se divide en tres unidades, las cuales son: 1. Introducción al diseño de sitios web, en esta vemos que es y cómo funciona el sistema de control de versiones GIT y el uso de GITHUB
para sincronizar nuestros proyectos a través de internet y así poder trabajar de forma colaborativa, 2. HTML5 y 3. CSS3. En las unidades 2 y 3 veremos el uso de HTML y CSS para desarrollar páginas web.

El OVI Diseño de sitios web está elaborado con el fin de dar a conocer información básica sobre el manejo de herramientas enfocadas al desarrollo web como los son GIT, GITHUB, HTML5 Y CSS3.
Esto se realizara a través de lecturas, videos y actividades cortas y fáciles de entender.
